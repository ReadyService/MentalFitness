# Video Game Coding with SpaceMiner ## Summary Students learn to build their own games with SpaceMiner and build fundamental skills in programming, using the world's most popular programming language, JavaScript. Students produce their own levels that they can further customize at home and play against friends and family. ## What students learn * Basics of JavaScript programming * String, Number, Boolean, Object, and Array data types * Conditional logic and looping with `if`, `else`, `for`, and `while` * Creating functions for making reusable code * Using free online code libraries ## What students produce * Their own game level * Working code * Learning assessment quiz results
